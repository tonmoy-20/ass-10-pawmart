# PawMart – Pet Adoption & Supplies Marketplace
# Live Website:https://paw-mart-enina.netlify.app/



PawMart is a modern pet adoption and supplies marketplace where users can adopt lovable pets, explore essential pet products, and manage their listings with ease.

# Features of the website

. Pet Adoption Listings — Browse pets available for adoption with detailed profiles.

. Add & Manage Listings — Users can add, update, and delete their own service/pet listings.

.Order System — Users can place orders for pets or supplies with instant confirmation.

. Secure Authentication — Login/Signup with Firebase authentication.

. Dynamic UI + Dark Mode — Fully responsive UI with theme toggle saved in Local Storage.

. Mobile-Friendly Navigation — Smooth dropdown menu and user-friendly mobile layout.

. Category-Based Browsing — Explore pets, food, accessories, and care products.

# Application use---

React 

React router

Firebase Authentication

Express.js Backend

MongoDB Database

Tailwind CSS + DaisyUI

Axios
